2017-01-12

Mobile first media queries

Non-mobile first media queries

Things to consider…

  • Don’t use only device sizes as breakpoints. Start at the smallest relevant screen size (320px)
  • Slowly start to scale up your screen
  • Once your design starts to become strained, add another media query to make any necessary changes to the layout
  • Continue until you reach the largest of screen sizes (2000px max - don’t need to go any higher than this really)
  • Try not to hide content on smaller screens. More and more users are viewing websites on mobile devices and require the same content as desktop users
