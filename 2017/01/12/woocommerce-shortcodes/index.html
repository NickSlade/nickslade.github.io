WooCommerce shortcodes

2017-01-12

Page shortcodes

  • [woocommerce_cart] - shows the cart page
  • [woocommerce_checkout] - shows the checkout page
  • [woocommerce_order_tracking] - shows the order tracking form
  • [woocommerce_my_account] - shows the user account page

These will usually be added to pages automatically generated by WooCommerce and should’t need to be added manually.

Cart

[woocommerce_cart]

Checkout

[woocommerce_checkout]

Order Tracking Form

[woocommerce_order_tracking]

My Account

You can specify the number of orders to show. By default, it’s set to 15 (use -1 to display all orders.)

Args:

[woocommerce_my_account order_count="12"]

Recent Products

The ‘per_page’ shortcode determines how many products to show on the page and the columns attribute controls how many columns wide the products should be before wrapping.

Args:

[recent_products per_page="12" columns="4"]

Featured Products

Works the same as recent products but displays products that have been set as “featured.” E.g. show 12 featured products in 4 columns.

Args:

[featured_products per_page="12" columns="4"]

Product

Show a single product by ID or SKU. Make sure it isn’t set to Hidden in the Catalog Visibility.
[product id="99"]
[product sku="FOO"]

Products

Show multiple products by ID or SKU.

Args:

[products ids="1, 2, 3, 4, 5"]
[products skus="foo, bar, baz" orderby="date" order="desc"]

Add to Cart

Args:

[add_to_cart id="99"]

Add to Cart URL

Args:

[add_to_cart_url id="99"]

Product Category

Show multiple products in a category by slug (found in WooCommerce > Products > Categories).

Args:

[product_category category="appliances"]

Product Categories

Display product categories loop

Args:

The number field is used to display the number of products and the ids field is to tell the shortcode which categories to display.

[product_categories number="12" parent="0"]
Set the parent parameter to 0 to only display top level categories. Set IDs to a comma separated list of category IDs to only show those.

To find the Category ID, go to the Product Categories screen, hover over the category and the ID appears in the URL.

Product Page

Show a full single product page by ID or SKU.

[product_page id="99"]
[product_page sku="FOO"]

Sale Products

List all products on sale.

Args:

[sale_products per_page="12"]

Best-Selling Products

List best-selling products on sale.

Args:

[best_selling_products per_page="12"]

Related Products

List related products.

Args:

[related_products per_page="12"]

Top Rated Products

List top-rated products on sale.

Args:

[top_rated_products per_page="12"]

via https://docs.woocommerce.com/document/woocommerce-shortcodes/
